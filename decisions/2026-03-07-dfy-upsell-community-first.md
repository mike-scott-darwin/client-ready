# Decision: DFY Upsell + Community-First Funnel Restructure

> **Update (2026-07-04):** The $97/mo price set here is now the *post-beta target*. The community is launching at **$47/mo beta entry** to seed the room first. See [2026-07-04-community-47-beta-entry.md](2026-07-04-community-47-beta-entry.md).

## What Changed

Replaced Sprint ($297 OTO) and Blueprint ($397 OTO) with:
- **DFY Offer Build ($197)** — AI-automated, Claude API generates ICP + offer doc + landing page copy + ad hooks from an 8-question form. Michael reviews (~10 min). Bundled with 30-day community trial.
- **DFY Lite ($97)** — Downsell. ICP + offer doc only. Same 30-day community trial.
- **Newsletter ($37/mo)** — "What's Working Now" continuity offer, replaces old OTO 3 Community position.

Community moves from buried downsell ($47/mo, conditional) to central engine ($97/mo, 30-day trial bundled with DFY). Month-to-month, cancel anytime — no annual lock-in.

Sprint curriculum and weekly calls move into community content. Blueprint splits: automatable parts → DFY upsell, high-touch parts → Accelerator exclusive.

Accelerator ($5K) sales channel shifts from cold email to warm community announcements.

## Why

- Day-1 AOV drops ~$25, but 90-day value per buyer increases ~$68 (35%)
- Recurring revenue per 100 buyers: $313/mo → $1,344/mo (4.3x)
- Accelerator conversion: ~0.5% (email) → ~2% (community) = $5K-$10K more per 100 buyers
- DFY upsell costs ~$0.15-0.30 per generation (Claude API) vs. hours of manual work
- Community solves the testimonial gap — members generating results = constant proof

## Positioning Reframe

Old: "No subscriptions. Sprint is one-time $297. Done."
New: "Month-to-month. Cancel anytime. You stay because the calls and templates are worth it — not because you're locked in."

This aligns with "test, validate, build" — stay flexible, no commitment you can't walk away from.

## Implementation Phases

1. Build DFY OTO + form + Claude API integration
2. Swap OTOs in checkout flow
3. Move Sprint content into community, set community price to $97/mo
4. Track and optimize (DFY take rate, trial conversion, Accelerator inquiries)
