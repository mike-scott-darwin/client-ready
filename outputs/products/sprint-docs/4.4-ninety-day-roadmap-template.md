### Weeks 5-6: Optimize

**Priority:** Apply what you learned in Week 4. Fix the bottleneck. Test new variations.

**Target:** Reduce CPA by 20-30% through testing.

**Action items:**

- [ ] Test 2 new landing page headlines

## Month 3: Scale or Expand

### Decision Point (Start of Month 3)

Answer honestly:

**Is the funnel profitable?** (Revenue > Ad Spend consistently for 2+ weeks)

- [ ] **YES** — Scale. More budget, new creative, expand reach.
- [ ] **BORDERLINE** — Keep optimizing. Don't increase spend yet. Focus on conversion rate.
- [ ] **NO** — Revisit the offer. The problem isn't the funnel — it's probably the offer or the audience.

### Weeks 9-10: Scale or Pivot

**If scaling:**

- [ ] Increase daily budget to: _______________
- [ ] Launch 5+ new ad variations (new hooks, new creative formats)
- [ ] Test video ads if you haven't yet
- [ ] Consider a second audience (interest-based if you've been running broad, or vice versa)
- [ ] Review: is there a pattern in who's buying? Use that to refine targeting.

**If pivoting:**

- [ ] Go back to validation (Module 1.4) — 5 new conversations
- [ ] Test a different angle on the same offer
- [ ] Consider adjusting price (up or down)
- [ ] Rewrite the landing page headline and problem section

### Weeks 11-12: Expand

**Priority:** Build the next layer of your business.

**Options (check the ones relevant to you):**

- [ ] Add a retargeting campaign for page visitors who didn't buy
- [ ] Launch a second front-end offer (use an existing bump as a standalone entry point)
- [ ] Build out the upsell/ascension sequence for existing buyers
- [ ] Create a content strategy to supplement paid traffic
- [ ] Explore a second traffic source (YouTube, TikTok, Google)

## Decision Triggers: When to Level Up

- **Profitable for 14+ days straight:** Funnel works. Time to grow. Scale ad spend 20% every 48 hours.
- **AOV consistently above 100:** Bumps are working. Test higher bump prices for more revenue.
- **Spending 100+ per day profitably:** Ready for professional strategy help. Consider the Accelerator ($5K done-with-you intensive).
- **Want to add more offers or expand:** System works, ready to build on it. Consider the Accelerator (8-week intensive).
- **Need ongoing accountability:** Don't want to go solo. Community membership (ongoing support).
