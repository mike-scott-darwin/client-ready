# Client Ready — Cash-Flow Relaunch Plan

**Goal:** Get the $47 funnel back to cash flow fast, using Miles's June-2026 cadence: launch **3 digital snacks at once → finish the test → double down on the winner.**

**Strategic choice:** All 3 snacks point at the **same validated $47 product** (Client Ready Offer System) but through **3 different pain doors** — each a distinct short-form page + headline-matched ads. This is the fastest launchable version of "3 snacks": one product, three entry pains. (Testing 3 *separate* front-end products — DM Scripts $37, One-Page Funnel $57 — is the later expansion once a door wins. See offer.md "Multiple entry points.")

---

## The 3 Snacks to Test

Each snack = one pain door → one headline → one short page → 5 headline-matched ads. The page's pre-headline + headline go *verbatim* into the ads (ad↔page match rule).

### Snack A — "Validate" door (solution-aware buyer)
- **Who:** Knows they need a clear offer, ready to act.
- **Page headline:** *"Validate your $5K+ offer before you build anything — in one afternoon."*
- **Mechanism on page:** 5 AI prompts → extract zone of genius → offer doc. (Straight from offer.md core promise.)
- **Backed by:** content-ideas #5 (Minimum Viable Offer), #2 (Stop surveying, start selling).
- **Why test it:** It's the core promise. Baseline.

### Snack B — "Confused → Clear" door (problem-aware buyer)
- **Who:** Has expertise, gives a different answer every time someone asks "what do you do?"
- **Page headline:** *"Give a different answer every time someone asks what you do? Fix your offer in one afternoon."*
- **Mechanism on page:** Same $47 product, framed as the confusion→clarity fix (offer.md "Confused → Clear → Converting").
- **Backed by:** offer.md transformation arc; audience "the 3am worry."
- **Why test it:** Hits an earlier, more emotional pain than "validate" — may convert colder traffic.

### Snack C — "Content treadmill" door (pain-first / unaware-of-cause)
- **Who:** Posted for months, made nothing, blames their content.
- **Page headline:** *"Posted for a year and made nothing? It's not your content. It's your offer."*
- **Mechanism on page:** Reframe — posting is downstream of a clear offer; validate the offer first. Same $47.
- **Backed by:** content-ideas #16 (72% of validations use paid traffic), #4 (free lead magnet lies); the "build order" framework video angle.
- **Why test it:** Pure pain-first hook, widest top of funnel. Often the cheapest CPA on cold traffic.

## Test Setup (per snack)

Identical mechanics across all three — only the door changes:

| Lever | Setting |
|---|---|
| Campaign | ABO, Sales objective, **ad-set budget** |
| Ad sets | Big Six ($10/day) + Worldwide ($10/day) — 2 ad sets to start |
| Creatives | 5 images × **5 different copies** (1:1 at $10/day), incl. 1–3 text-heavy ads with the page headline verbatim |
| CTA | "Learn more" |
| Spend per snack | **$100–300 total**, finish the full test (sales cluster first/last day) |
| Meta toggles | All auto-test OFF (multi-advertiser, personalized destinations, dynamic copy, "combine ads") |

**Decision gate:** A good snack makes a sale by **~$70–100**. Pause losers. Double down on the winner(s) — often 2 of 3 sell.

---

## After a Snack Wins (optimization order — do NOT skip ahead)

1. **Order bumps → 33%+** (fix copy or swap the bump). 25% = keep; 33%+ = goal.
2. **Build 1–2 OTOs** — only now (front end is validated). DFY Offer Build $197 is OTO 1; Newsletter $37/mo is OTO 2.
3. **A/B test the front-end headline** (screenshot page → AI B-variant → split).
4. **Then scale budget** — and only once OTO converts **5–10%**. A $113 CPA is fine if bumps+OTOs carry AOV.

## Pre-Launch Checklist (must exist before $1 of spend)

- [ ] **3 short-form pages live** (one per door), each passing the 5-Second Test
- [ ] **Two-step order form** on every page (opt-in → bumps) — mandatory for abandoned-cart + list
- [ ] **3 order bumps** wired ($37 DM Scripts / $67 Templates / $97 Playbook)
- [ ] **CAPI + pixel** firing correctly (one client lost their pixel to a currency/geo glitch — verify it's active and in USD)
- [ ] **Daily email** ready to send from day one — reply-based, every email an offer (~2.4% reply benchmark)
- [ ] **Bio lead magnet** ("The Build Order" one-pager) feeding the welcome sequence — for organic/TikTok traffic in parallel
- [ ] 5 headline-matched ads drafted per snack (15 total) — AI-generated copy + images, keep the ugly ones

## Suggested Timeline

- **Days 1–3:** Build 3 pages (clone the $47 page, swap headline + opening per door), wire bumps + two-step form, verify pixel/CAPI.
- **Day 3:** Draft 15 ads (5/snack) with AI. Launch all 3 snacks, $20/day each ($10 Big Six + $10 worldwide).
- **Days 4–10:** Don't touch. Let each finish ~$100–300. Send daily emails to any buyers.
- **~Day 10:** Read results. Pause losers. On the winner: bumps → OTOs → headline test → scale.
