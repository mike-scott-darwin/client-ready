# Ad Launch Strategy — Client Ready $47 Offer

## The 4 Angles (One Ad Set Each)

| Ad Set | Angle | Hook | Use | Awareness Level |
|--------|-------|------|-----|-----------------|
| 1 | **Before the Funnel** | "Everyone's selling you funnels. Nobody's asking: do you have an offer worth building one for?" | Cold — positioning | Problem Aware → Unaware |
| 2 | **Misalignment** | "I've built offers that made money and made me miserable." | Warm + cold — trust | Unaware (belief shift) |
| 3 | **Content Merry-Go-Round** | "You've been posting for 12 months. Getting likes. Nobody's buying." | Cold — pain point | Problem Aware |
| 4 | **Clarity Unlock** | "The answer was already inside them. They just couldn't see it." | Cold + warm — transformation | Solution Aware |

**Awareness spread matters.** Cat Howell: *"Ads die after week 1 because Facebook shows to warm audience first. Once exhausted, it shifts to cold. If you only have 'aware' creative, your ads will tank."* These 4 angles cover all levels — when Before the Funnel and Misalignment perform, you're reaching the unaware market that scales.

---

## Campaign Structure: Lean Start → Sequential Scale

Launch lean. Validate one angle. Fund the next angle with revenue.

| Stage | When | Budget | Structure |
|-------|------|--------|-----------|
| **Stage 1: Validate** | Day 1 (launch) | $30/day | 1 ad set (Before the Funnel), 3-4 ads, broad targeting |
| **Stage 2: Expand** | After 10+ sales, CPA confirmed | $60/day | Add angle 2 (Content Merry-Go-Round) |
| **Stage 3: Scale** | After 30 sales (validated) | $90/day | Add angle 3 (Clarity Unlock) |
| **Stage 4: Full** | After AOV confirmed $100+ | $200/day | Add angle 4 (Misalignment) or graduate winners to CBO |
| **CBO Winners** | After 10+ sales per creative | Shift budget | Graduate winners via Post ID extraction |
| **ASC Scaling** | After 100+ sales, matured pixel | — | Final stage only |

**Why sequential (not all 4 at once):**
- $30/day can't support 4 ad sets — each would get $7.50/day, not enough signal
- One angle at $30/day = each ad gets ~$8-10/day — clean data within a week
- Revenue from Stage 1 funds Stage 2. Each stage pays for the next.
- $20 rule still applies — at $30/day you know within 2-3 days what's working

**Why Before the Funnel first:**
- Strongest positioning angle — differentiates from every other coach offer ad
- Bridges Problem Aware → Unaware — reaches the widest cold audience
- USV candidate — "Everyone's selling you funnels" is the most contrarian hook

**Cat Howell alignment:** Her lean start method is $25-30/day, one angle, 3-4 ads. Validate it, then add angles as budget grows. CBO comes later when you have proven winners and $100+/day to give the algorithm enough data.

**The graduation path:**
1. Run Stage 1 for 72 hours untouched
2. At $20 spend per ad, anything with zero signal (no clicks, no adds to cart) — kill that image, keep the copy block
3. At 10+ sales — CPA is confirmed, add Stage 2 angle
4. At 30 sales — validated. Add Stage 3 angle.
5. At $100+ AOV confirmed — go to $200/day (Stage 4) or graduate winners to CBO
6. CBO: one ad set, 5-6 proven winners, Advantage Plus targeting, $100/day starting budget
7. Scale CBO by adding more winning creative (Cat: *"Every time you raise the budget, you should be adding more ads into your ad set."*)

## Creative Count Per Ad Set

Each ad set = one angle. Within each:

| Element | Count | Details |
|---------|-------|---------|
| Copy blocks | 1 | Best primary (P1) — keep it lean so each ad gets enough spend |
| Images | 3 | Ugly static, product reveal, before/after |
| Video creatives | 1 | Silent review or B-roll with text overlay |
| **Total ads per ad set** | **4** | 3 image + 1 video |

At $30/day with 3-4 ads, each ad gets ~$8-10/day. Cat Howell's math: at $2 CPC, need $100 spend (~12 days per ad) to expect one sale. **Lean ad sets = faster data.**

## Creative Production Priority

Build in this order — fastest to produce, highest-performing format first:

| Priority | Format | Time | What to Make |
|----------|--------|------|-------------|
| 1 | Ugly static | 15 min each | One-liners as Notes app screenshots, tweet formats — 20 of these covers half your images |
| 2 | Product mockup static | 30 min each | Device screens showing the AI prompts, offer document template |
| 3 | Silent Review video | 1-2 hrs each | Screen recording scrolling through the system with reaction shots + text overlay |
| 4 | B-roll + text overlay | 1-2 hrs each | Footage of working + text overlay with hooks |

**Fastest launch path:** Produce 3 static images + 1 video for Before the Funnel. Can be done in one session (under 2 hours). You're live the same day.

## Targeting

- Broad is default — creative IS targeting (post-Andromeda)
- Advantage Plus targeting — no interests, no lookalikes, no exclusions
- Target by country only: US, UK, Canada, Australia, New Zealand
- One ad set worldwide (broader, cheaper CPMs) vs one high-income only — compare
- No separate retargeting campaigns — the algorithm handles it within broad

## Kill Criteria

| Signal | Action |
|--------|--------|
| $20 spend, zero clicks | Kill that image creative |
| $20 spend, clicks but no add-to-cart | Kill that copy block |
| CPA > $150 after 72 hrs | Kill the ad set |
| CPA $80-120 | Let it run, monitor |
| CPA < $80 | Scale 20% every 48 hrs |
| Frequency 2.0 | Warning — monitor closely |
| Frequency 3.0+ | Fatigued — do NOT scale, roll out new creatives |

**Cat Howell on frequency:** *"If you try to scale a campaign with fatigued ads, you're gonna blow that up."* Check frequency BEFORE scaling any winner.

## Ad Setup Rules (Cat Howell)

These apply to every ad created:

- **CTA button:** "Learn More" (not "Shop Now" — low-ticket performs better with soft CTA)
- **Advantage+ Creative:** Turn OFF
- **Creative enhancements:** Turn OFF all (except maybe music)
- **Site links:** Turn OFF
- **Conversion event:** Purchase (not Add to Cart, not Lead)
- **Placements:** Advantage+ Placements (NOT manual "Feed Only")

**Ad naming convention:** `[Awareness Level] - [Format] - [Angle]`
Examples:
- `Problem Aware - Ugly Static - Before the Funnel`
- `Unaware - Silent Review - Misalignment`
- `Solution Aware - B-roll - Clarity Unlock`

## Target Metrics

| Metric | Target | Source |
|--------|--------|--------|
| Front-end AOV | $90-110 | Cat Howell: $100+ before scaling |
| Bump take rate | 50%+ (any bump) | Cat Howell: "50% should be buying bumps" |
| CPA (break even) | < $100 | Cat: $50-100 normal at 2% CVR, $2 CPC |
| CPA (profitable) | < $80 | — |
| Checkout CVR | 30% | — |
| Landing page CVR | 2-5% cold | Cat: below 2% = messaging problem |
| CPC | Under $7 | Cat: higher OK for niche B2B |
| CTR | Over 1% | Cat: below = hook problem |
| ROAS | 2.0+ | Cat's range: 2.0-2.35 |

**Cat's validation thresholds:**

| Sales | Status |
|-------|--------|
| 5-10 | "Onto something" but NOT validated |
| 30 | Minimum to trust split test data |
| 100 | Truly validated — numbers stabilize |

**Don't scale until AOV is $100+ consistently.** Cat: *"Don't scale ads until AOV is solid over $100."* If bump take rates are low, fix bumps before increasing ad spend.

### Troubleshooting Checklist (Cat Howell's 8-Point)

If ads aren't converting, check in this order:

1. **Enough spend at ad level?** At 2% CVR, $2 CPC → need $100 per expected sale
2. **Enough landing page views?** At 2% CVR → need 50 views per expected sale
3. **CPC acceptable?** Target under $7
4. **CTR acceptable?** Target over 1% — below = hook isn't stopping the scroll
5. **Ad "keeps the scent"?** Landing page matches what the ad promised
6. **Messaging & positioning?** Clicks but no sales = sales page losing the job
7. **Pricing?** $17-$67 sweet spot cold. $47 is confirmed.
8. **Do you actually care?** Cat: *"If it's not a song of your soul... you will give up."*
