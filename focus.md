## Key Insights

### Cat Howell (Low-Ticket Ecosystem)
- One funnel with 3 offers can be profitable standalone
- Target AOV: $100-$200
- Problem-aware and unaware = biggest markets
- Sell first, build later

### Andromeda Update (Meta Ads 2026)
- **Creative IS targeting** — need 15-20+ diverse creatives
- 70% of success = creative (stops scroll in 1.5 sec)
- Ads across awareness spectrum (unaware → problem → solution → product)
- Use Advantage+ Sales campaigns, not interest targeting
- Short copy (<125 chars) outperforms long-form on cold
- 5-second rule: value prop in first 5 seconds or deprioritized
