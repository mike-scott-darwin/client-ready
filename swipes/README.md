# Email Swipes

Competitor emails for style/format analysis.

## How to Add

1. **PDF**: Save email as PDF, drop here
2. **Markdown**: Copy email content into `.md` file using template below

## Folder Structure

```
swipes/
├── welcome/       # Welcome sequences
├── sales/         # Sales/promo emails
├── nurture/       # Value/content emails
└── abandoned/     # Cart abandonment
```

## Template

```markdown
---
from: Competitor Name
subject: Original subject line
type: welcome | sales | nurture | abandoned
date: YYYY-MM-DD
---

# Subject: [Original subject line]

[Paste email content here]

---
```
