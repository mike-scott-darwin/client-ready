# Parallel Agents Live Demo Concept

**Context:** Presentation demo showing the power of reference files + parallel agents for content production.

## Summary

Live demo where one command spawns 4 agents simultaneously, producing a full week of multi-platform content in 60-90 seconds for $3-8. The reference files are the moat -- the demo makes that tangible.

## Demo Flow

### Setup (30 seconds)
- Show the reference files briefly (soul.md, voice.md, offer.md, content-strategy.md)
- "This is what you build once. It's who you are, what you sell, how you sound."

### Optional Contrast (30 seconds)
- Type "write me a LinkedIn post about coaching" into ChatGPT
- Show the generic AI slop output
- "This is what everyone's doing. No context. No voice. No system."

### The Batch (60-90 seconds)
- Type one command: "batch me next week's content"
- Four terminal spinners appear simultaneously:

```
⠋ Agent 1: Writing Tuesday newsletter...
⠋ Agent 2: Writing 5 LinkedIn posts...
⠋ Agent 3: Writing 2 X threads (10 tweets)...
⠋ Agent 4: Writing 20 daily tweets...
```

- Audience watches them complete one by one
- Open the files -- real content, correct voice, ready to publish

### The Reveal
- Show the content quality (sounds like you, not AI slop)
- Show the cost: "That was about $5 in API tokens"
- Show the time: "90 seconds for a full week across 3 platforms"

## Why It Lands

1. **It's not fake** -- live Claude Code session, not a slide deck. Real agents, real output, real time.
2. **The volume is visceral** -- one command → a full week of content across 3 platforms. People can feel what that saves.
3. **The quality is the surprise** -- they expect AI slop. They get content that sounds like you -- because the agents read your reference files.
4. **The cost reveal** -- after the content appears: "$5 in tokens." Audience does the math vs hiring a content team.

## Key Messaging

- "Everyone's chasing the perfect prompt. The answer is a system, not a sentence."
- "Build the context once. Run it forever."
- "Reference files are the moat. AI is the amplifier. Feed it chaos, get polished chaos back. Feed it your context, get content that sounds like you."
- "That's a week of content across 3 platforms. Under $8. In 90 seconds. Because the reference files already know who you are, what you sell, and how you sound."
