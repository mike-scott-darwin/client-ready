## Key Strategic Insights

### 1. Frame Recurring as "Sprint"

Instead of "$37/mo (cancel anytime)" → "4-Week Sprint for $297"

| Option | Price | Frame |
|--------|-------|-------|
| Quick Win | $97 | "1 week access" |
| Full Sprint | $297 | "4 weeks to achieve X result" |

### 2. $97 Order Bump = 47% Pickup

**Rule:** If bump conversion >40%, test higher prices.

### 3. OTO Sequencing Logic

Each OTO solves the NEXT problem:

```
Snack Purchase → "What new problem does this create?"
     ↓
OTO 1 solves that problem
     ↓
OTO 2 solves the next problem
     ↓
Backend = "do it all for me"
```

### 4. $300 Validation Test

Before building a full funnel:
1. Create minimal snack offer
2. Build simple landing page
3. Run $300 in ads over 3-7 days
4. Target: 2-3 sales = validated
5. If validated → scale to $300-400/day
6. If no sales → kill it

### 5. Order Bumps = Energy Reducers

| Good Bumps | Bad Bumps |
|------------|-----------|
| Templates | More courses |
| AI tools | More reading |
| DFY audits | More modules |
| Swipe files | More complexity |

### 6. Email Sequence: Lead to Limitations

| Email | Purpose |
|-------|---------|
| 1-3 | Deliver value, help them use product |
| 4-5 | Surface limitation — "Now you need X..." |
| 6-7 | Position call as the solution |

**Target:** 3-5% call booking rate.
